# NLW5 - Trilha React - Podcastr
Aplicação feita durante o evento Next Level Week 05 da [Rocketseat](https://rocketseat.com.br/) de 19/04/2021 - 25/04/2021. Podcastr é uma aplicação web feita em React com Next.js para geração de páginas estáticas.
<p>Ao acessar a aplicação, o usuário verá os dois podcasts mais recentes em destaque, enquanto o restante estará listado logo abaixo. É possível escutar um podcast, deixá-lo em loop e aleatorizar o próximo podcast da lista. O podcast que está sendo escutado é mostrado na barra lateral da aplicação.</p>
<p>Ao clicar no título do podcast, o usuário navegará para uma página contendo informações adicionais do podcast.</p>

# Utilizar projeto
## Requisitos
    Instalar o Git
    https://git-scm.com/downloads

    Instalar o NodeJs
    https://nodejs.org/en/

    Instalar JSON server
    https://www.npmjs.com/package/json-server
## Primeiro passo - baixar o projeto
Faça download do repositório acima ou use o seguinte comando na pasta onde deseja baixar o projeto:
 - git clone https://github.com/AloisioFernandes/NLW5-React.git

## Segundo passo - instalar dependências
Digite o seguinte comando dentro da pasta podcastr através do terminal:
 - npm install

## Terceiro passo - iniciar servidor JSON
Ainda na pasta podcastr, digite o seguinte comando no terminal:
 - npm run server

## Quarto passo - iniciar projeto
Acesse a pasta podcastr por outro terminal e digite o seguinte comando:
 - npm run dev

## Quinto passo - acessar projeto
Na aba do navegador, acesse o endereço: http://localhost:3000/
